# dPaste - Decentralized P2P Pastebin

A fully decentralized Pastebin using P2P for data storage.
dPaste is powered by the [gundb](https://gun.eco) library and written in React.

## Setup client

```
pnpm install
pnpm start

pnpm build
```

## Setup own 'gun' relay server

```
node relay.js
```
